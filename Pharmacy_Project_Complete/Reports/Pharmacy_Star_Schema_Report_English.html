📊 Star Schema Report

Pharmaceutical Chain Sales Data Warehouse

Comprehensive Analysis and Dimensional Modeling

🎯 Project Overview

This project focuses on designing and implementing a multidimensional data model (Star Schema) for a pharmaceutical chain. The goal is to provide a foundation for sales analysis, customer pattern recognition, and managerial decision-making.

Project Objectives:
  • Design Star Schema for pharmacy sales transactions
  • Create Dimension Tables and Fact Table
  • Develop Pivot Tables for multidimensional analysis
  • Generate comprehensive analytical reports
  • Extract business insights from transaction data

🏗️ Star Schema Architecture

The Star Schema is a dimensional data model used in OLAP systems. It consists of a central Fact Table and multiple Dimension Tables.

Model Components:

Table Name Type Records Description
FACT_SALES Fact Table 5,000 All pharmaceutical sales transactions
DIM_TIME Dimension 366 All days of year 2024
DIM_CUSTOMER Dimension 4,347 Customer demographic information
DIM_DRUG Dimension 20 Drug catalog (20 types)
DIM_BRANCH Dimension 10 Pharmacy branches
✓ Advantages of Star Schema:
  • Faster queries due to fewer joins
  • Easier understanding of data structure
  • Optimized for analytical (OLAP) queries
  • High scalability for adding new dimensions

📋 Dimension Tables

1) Time Dimension (DIM_TIME)

Covers: Date, Day, Month, Quarter, Year, DayOfWeek, WeekOfYear

Purpose: Temporal analysis of sales (daily, monthly, quarterly)

2) Customer Dimension (DIM_CUSTOMER)

Covers: CustomerID, Gender, AgeGroup

Age Groups: <18, 18-35, 36-50, 51-65, >65

Purpose: Customer segmentation and behavior analysis

3) Drug Dimension (DIM_DRUG)

Covers: DrugCode, DrugName, Category

Categories: 13 pharmaceutical drug categories

Purpose: Sales analysis by drugs and categories

4) Branch Dimension (DIM_BRANCH)

Covers: BranchCode, BranchName

Count: 10 branches (B001 to B010)

Purpose: Revenue comparison across pharmacy units

📊 Fact Table (FACT_SALES)

The Fact Table contains all sales transactions. Each row represents a purchase.

Measure Columns:

Column Data Type Description
Quantity Integer Units of drugs sold (1-4)
UnitPrice Decimal Price per unit (Iranian Rials)

🔄 Pivot Tables (7 Analysis Tables)

Pivot tables enable multidimensional analysis across different dimensions:

  1. Monthly Revenue Trend - Revenue by month
  2. Branch vs Drug Category - Sales per branch by category
  3. Age Group vs Gender - Customer demographic patterns
  4. Payment Method by Branch - Payment preference analysis
  5. Day of Week Analysis - Daily sales patterns
  6. Top 10 Drugs - Best-selling pharmaceutical products
  7. Age Distribution - Customer age group analysis

💡 Business Insights & Recommendations

Key Findings:

1. Revenue Distribution by Branch:
  • Branch 5 generates 26% of total revenue (186 Billion)
  • Branch 10 generates only 8.8% of total revenue (148 Billion)
  • Recommendation: Investigate Branch 10 efficiency
2. Customer Behavior Patterns:
  • Age group 36-50 is the largest purchaser (344 Billion)
  • Gender distribution nearly equal (50%-50%)
  • Recommendation: Target marketing campaigns for 36-50 age group
3. Payment Method Preferences:
  • Cash: 40% (647 Billion)
  • Card: 35% (593 Billion)
  • Insurance: 20% (349 Billion)
  • Check: 5% (83 Billion)
  • Recommendation: Promote digital payment methods
4. Best-Selling Drugs:
  • Antibiotic (D0005): 182 Billion (Highest)
  • Anticonvulsant (D0013): 129 Billion
  • Statin (D0010): 126 Billion
  • Recommendation: Ensure adequate inventory for top drugs

🎓 Conclusion

This project demonstrates how Star Schema enables comprehensive analysis of pharmaceutical sales data. By designing proper data structure and creating multiple pivot tables, organizations can:

  • 💰 Gain deeper understanding of revenue sources
  • 📊 Analyze sales patterns across different dimensions
  • 👥 Examine customer behavior by demographics
  • 📈 Make data-driven managerial decisions
  • 🎯 Execute targeted marketing campaigns
Key Takeaway: Star Schema models are powerful tools for analysis and reporting in business systems. They provide clear data structure, fast queries, and flexible pivot tables.
