- ✓ طراحی مدل Star Schema برای فروش داروخانه

## 📊 معماری Star Schema

```
                    ┌─────────────────────────┐
                    │   FACT_SALES            │
                    │  ─────────────────────  │
                    │  • TransactionID        │
                    │  • DateKey (FK)         │
                    │  • CustomerID (FK)      │
                    │  • DrugCode (FK)        │
                    │  • BranchCode (FK)      │
                    │  • Quantity             │
                    │  • UnitPrice            │
                    │  • TotalAmount          │
                    │  • PaymentMethod        │
                    └─────────────────────────┘
                            ↓↓↓↓
        ┌───────────────────┼───────────────────┐
        │                   │                   │
    DIM_TIME          DIM_CUSTOMER        DIM_DRUG
   ────────────      ──────────────      ─────────
   • Date (PK)       • CustomerID (PK)   • DrugCode (PK)
   • Day             • Gender            • DrugName
   • Month           • AgeGroup          • Category
   • Quarter                             
   • Year                DIM_BRANCH
   • DayOfWeek      ──────────────────
   • WeekOfYear     • BranchCode (PK)
                    • BranchName
```

## 📋 توضیح جداول

### 1. جدول زمان (DIM_TIME)
- **رکورد‌ها:** 366 (تمام روزهای سال 2024)
- **ستون‌ها:** Date, Day, Month, Quarter, Year, DayOfWeek, WeekOfYear
- **کاربرد:** تحلیل فروش بر حسب زمان

### 2. جدول مشتری (DIM_CUSTOMER)
- **رکورد‌ها:** 4,347
- **ستون‌ها:** CustomerID, Gender, AgeGroup
- **گروه‌های سنی:** <18, 18-35, 36-50, 51-65, >65
- **کاربرد:** تقسیم‌بندی مشتریان

### 3. جدول دارو (DIM_DRUG)
- **رکورد‌ها:** 20
- **ستون‌ها:** DrugCode, DrugName, Category
- **دسته‌های دارویی:** 13 دسته مختلف
- **کاربرد:** تحلیل فروش به تفکیک دارو

### 4. جدول شعبه (DIM_BRANCH)
- **رکورد‌ها:** 10
- **ستون‌ها:** BranchCode, BranchName
- **کاربرد:** تقسیم درآمد بین شعبه‌ها

### 5. جدول واقعیت (FACT_SALES) ⭐
- **رکورد‌ها:** 5,000 تراکنش
- **اندازه‌های اصلی:**
  - Quantity: تعداد واحد
  - UnitPrice: قیمت واحد
  - TotalAmount: مبلغ کل
  - PaymentMethod: روش پرداخت

### 7 Pivot Table ایجاد‌شده:

2. **شعبه در برابر دسته دارو** - فروش هر شعبه به تفکیک دسته‌بندی
3. **گروه سنی در برابر جنسیت** - فروش بر حسب سن و جنسیت
4. **روش پرداخت در شعبه‌ها** - تفضیل روش‌های پرداخت
5. **تحلیل روز هفته** - درآمد بر حسب روز هفته
6. **10 دارو برتر** - داروهای پرفروش
7. **توزیع سنی مشتریان** - درآمد بر حسب گروه سنی

## 🛠️ نحوه استفاده

### 1. باز کردن فایل‌های CSV
```python
import pandas as pd

# خواندن جداول ابعاد
dim_time = pd.read_csv('Data/dim_time.csv')
dim_drug = pd.read_csv('Data/dim_drug.csv')
dim_customer = pd.read_csv('Data/dim_customer.csv')
dim_branch = pd.read_csv('Data/dim_branch.csv')

# خواندن جدول واقعیت
fact_sales = pd.read_csv('Data/fact_sales.csv')
```

### 2. ایجاد Pivot Table
```python
# درآمد بر حسب ماه
pivot_month = fact_sales.groupby('Month')['TotalAmount'].sum()

# درآمد بر حسب شعبه
pivot_branch = fact_sales.groupby('BranchName')['TotalAmount'].sum()
```
